# coin-flip-app
COIN TOSS SIMULATOR

A simple and interactive Coin Toss Simulator built using Python and
Tkinter. This project simulates a real coin flip while keeping track of
statistics such as total tosses, heads count, and tails count.

FEATURES - Simple and user-friendly GUI - Random coin toss simulation -
Displays Heads or Tails result - Tracks total tosses - Maintains
separate counts for Heads and Tails - Reset button to clear all
statistics - Modern interface built with Tkinter

TECHNOLOGIES USED - Python 3 - Tkinter (GUI Library) - Random Module

HOW TO RUN

1.  Make sure Python 3 is installed on your system.

2.  Download or clone this repository.

3.  Open a terminal in the project folder.

4.  Run the following command:

    python coin.py

REQUIREMENTS - Python 3.x - Tkinter (usually included with Python
installations)

PROJECT STRUCTURE

coin.py -> Main application file README.txt -> Project documentation
